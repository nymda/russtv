# Russtv

SSTV encoder written in rust.  
See [WinSSTV](https://github.com/nymda/WinSSTV) and [CLSSTV](https://github.com/nymda/ClSSTV).  

**supported SSTV methods:**  
-Robot BW8  
-Robot BW16  
-Robot 24  
-Robot 36  
-Robot 72  
-Scottie 1  
-Scottie 2  
-Scottie DX  
-Martin 1  
-Martin 2  
-PD50, 90, 120, 160, 180, 240, 290  
-AVT90
