# Russtv

SSTV encoder written in rust.  
Basically a rewrite of the core of https://github.com/nymda/WinSSTV  

**supported SSTV methods:**  
-BW8  
-BW16  
-Robot 24  
-Robot 36  
-Robot 72  
-Scottie 1  
-Scottie 2  
-Scottie DX  
-Martin 1  
-Martin 2  
-PD50, 90, 120, 160, 180, 240, 290  
-MR73, 90, 115, 140, 175 (buggy)  
-AVT90
